Addon Developer Tools, Packages, and Documentation

Here you'll find a bunch of resources for creating Addons for the videogame Wildstar.

This website is a git repository that hosts all of the tools born out of WildStar NASA (Nexus Addon Standards Association), which is an Addon developer community trying to make Addon development as accessible as possible. One of the main NASA projects is Gemini, a set of open source Addons, Packages, and other tools for developers.
